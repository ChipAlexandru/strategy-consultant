Worker

Research. Analysis. Deliverable. Action.

Worker handles 90% of the analytical workload in any structured company task: strategy engagements, market assessments, competitive reviews, M&A diligence, operational analyses.

You focus on the 10% that creates unique value: steering, judgment, proprietary insight, and final go.

The plugin started in consulting and that remains its sharpest edge. But the same approach applies across most structured knowledge work.

How you'll work with it

1
Define the problem
Worker sharpens your question into a decision-oriented problem statement, asks what internal data or reference material you have, and confirms the deliverable format before research begins.
2
Multiple agents work independently
Two analysts work independent angles while a third cross-checks for consistency and source quality, producing a single validated research base you can review and add context to before analysis continues.
3
Everything gets pressure-tested
A sense-check agent produces a written report covering claim quality, counter-narratives, and math. The synthesis that follows builds a structured argument with a governing message and evidence map.
4
You get a structured deliverable
Word, Excel, or PowerPoint depending on what you confirmed upfront, with every claim sourced. A separate validation agent audits the output against the research base before it reaches you.

Get started

Download and upload to Cowork
  1. Download and unzip the plugin folder
  2. Open the Claude desktop app, switch to Cowork
  3. Click Customize, then Browse plugins, then Upload plugin
  4. Select the unzipped folder
Download plugin
Using the plugin

Once installed, the plugin activates automatically when Cowork recognises a relevant task. You can also ask Cowork to use it directly, for example: "Use Worker to analyse this market entry opportunity."

For the technically inclined: the full source, prompt architecture, and agent definitions are on GitHub. Contributions welcome.
