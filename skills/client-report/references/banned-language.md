# Banned Language Reference

Words and phrases that are prohibited in client deliverables, with their replacements.

## Banned Words and Replacements

| Banned | Replace With |
|--------|-------------|
| Leverage | Use, apply, build on |
| Synergies | [Specify: "$20M in combined purchasing savings" or "shared customer base of 5M"] |
| Unlock value | [Specify: "increase EBITDA by $30-40M through X"] |
| Best-in-class | [Name the company]: "[Company] achieves 25% margins vs. industry average of 18%" |
| Holistic | Comprehensive, covering [list what is included] |
| Ecosystem | Network, market, system (unless literally biology) |
| Paradigm shift | [Describe what changed]: "Customer preference shifted from ownership to subscription" |
| Move the needle | [Specify]: "Improve retention by 3-5 percentage points" |
| Low-hanging fruit | [Name it]: "Renegotiating the three largest supplier contracts" |
| Deep dive | Detailed analysis |
| Going forward | Over the next [time period] / From Q3 2026 onward |
| At the end of the day | [Delete entirely] |
| Thought leadership | [Delete entirely] |
| Streamline | Simplify, reduce steps from X to Y, cut processing time by Z% |
| Optimize | Improve [specific metric] by [amount] |
| Robust | Strong, reliable, tested — or [describe what makes it robust] |
| Scalable | Can handle [X] volume at [Y] marginal cost |
| Actionable | [Just make it actionable — don't label it] |
| Drill down | Examine, analyze in detail |
| Circle back | Follow up, revisit on [date] |
| Bandwidth | Capacity, staff hours, available time |
| Pivot | Change direction to [specific new direction] |
| Disrupt/disruptive | [Describe the specific change and its impact] |
| Value proposition | What the customer gets: [specific benefit] |
| Key takeaway | [Just state the takeaway] |
| Stakeholder alignment | Agreement from [name the specific people or groups] |
| Touch base | Meet, discuss, review |
| Take offline | Discuss separately |
| Boil the ocean | [Describe the scope problem specifically] |

## Banned Sentence Structures

| Banned Pattern | Replace With |
|---------------|-------------|
| "It is important to note that..." | [Just state the thing] |
| "It should be noted that..." | [Just state the thing] |
| "It goes without saying that..." | [Then don't say it, or just state it directly] |
| "As previously mentioned..." | [Restate briefly if needed, or just continue] |
| "In today's rapidly evolving landscape..." | [State the specific change] |
| "In order to..." | "To..." |
| "The fact that..." | [Delete and restructure] |
| "It is clear that..." | [State the conclusion directly] |
| "There is a need to..." | "[Subject] should..." |
| "We would recommend..." | "We recommend..." |

## The Clarity Test
Before finalizing any paragraph, ask: "Could I explain this to a smart 16-year-old?" If not, the language is too abstract. Rewrite with concrete nouns, specific numbers, and active verbs.
